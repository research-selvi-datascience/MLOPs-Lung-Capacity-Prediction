# MLOPs-Lung-Capacity-Prediction

LungCap Dataset

The LungCap dataset is a comprehensive collection of data that provides insights into the lung capacity of individuals. This dataset consists of 725 valid observations and includes five variables: Age, Height, Gender, Smoke (yes/no), and LungCap.

Key Variables

Age: The age of the individual.

Height: The height of the individual.

Gender: The gender of the individual (0 for female, 1 for male).

Smoke: Indicates whether the individual smokes (0 for no, 1 for yes).

LungCap: Forced Expiratory Volume, a measure of lung capacity.

The dataset can be obtained from Kaggle at the following link:  https://www.kaggle.com/datasets/jacopoferretti/lung-capacity-of-kids
